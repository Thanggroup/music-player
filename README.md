# Music Player

A simple JavaScript music player built step-by-step while learning web development.

## Features

- Play / Pause
- Next / Previous
- Playlist
- Progress bar
- Song time display

## Project Structure

assets/ – audio files  
v0.6 – basic player  
v0.7 – progress bar  
v0.8 – time display  

## Technologies

- HTML
- CSS
- JavaScript
