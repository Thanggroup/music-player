# Music Player

A simple JavaScript music player built step-by-step while learning web development.

## Features

- Play / Pause
- Next / Previous
- Playlist
- Progress bar
- Song time display

## Project Structure

assets/ – audio files  
v0.6 – basic player  
v0.7 – progress bar  
v0.8 – time display  

## Technologies

- HTML
- CSS
- JavaScript

# Mondi Player — Setup Guide

## Requirements

* Node.js (v18+ recommended)
* npm
* Android Studio

## Setup

```bash
npm install
npx cap sync
npx cap open android
```

## Run

* Open Android Studio
* Click Run ▶️

## Notes

* Web app lives in `/app`
* Do not modify `/android` unless needed
